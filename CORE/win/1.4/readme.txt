INSTALL
-------
  sudo dpkg -i aisf-core_1.4-1_amd64.deb

Installs to /usr/local/bin/aisf-core with desktop and app menu entries.
No additional files required. Uninstall: sudo dpkg -r aisf-core


LAUNCH
------
Run from the application menu (Utility category) or from a terminal:
  aisf-core


BASIC USAGE
-----------
1. Type your text in the editor
2. Click Submit (ALT+S) -- adds a UTC timestamp and copies to clipboard
3. Switch to your AI platform (ALT+TAB), paste and send (CTRL+V, ENTER)
4. Return to AISF (ALT+TAB), click New (ENTER) for next input

Session startup (recommended):
  After your first Submit, click Structure then Stabilize -- each sends
  a one-time setup block to the AI that improves output quality and
  reduces hallucinations.
